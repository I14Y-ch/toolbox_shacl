XSD Import Information

This import type transforms XML Schema Definition files into SHACL shapes. The converter handles:

  • Complex and simple type definitions
  • Element constraints and restrictions
  • Enumeration values
  • Type unions and extensions
  • Sequence and choice compositors
  • Attribute definitions

The resulting SHACL shapes will preserve the structure and constraints defined in your XSD file.
